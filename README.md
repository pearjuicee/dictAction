It is a dictation tool where users can choose to submit a text to practice their dictation or choose a stored dictation 
available on the website for different levels of proficiency. The tool will then read the text at the pace chosen by the
user and they can choose to either type it on the website or write it by hand on a paper. In the case that the user types 
it on the website, the tool will correct and grade the user's answer. Otherwise, the user can see the dictated text and 
correct his answer manually.
